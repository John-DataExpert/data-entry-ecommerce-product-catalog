# E-Commerce Product Catalog Creation

### Overview
This project simulates the task of building a complete product catalog for a new e-commerce store. It involved collecting data for 200 products from various digital source files and compiling it into a single, structured, and import-ready spreadsheet.

***Confidentiality Note:*** *The product names, SKUs, and prices are fictional and created for this portfolio piece. This project demonstrates my process for handling such tasks while respecting client data confidentiality.*

---

### The Problem
An e-commerce client needed to populate their online store with 200 products. The product information (name, category, price, description) was scattered across multiple supplier documents and web pages, lacking a consistent format. The task required meticulous data collection and entry to ensure the final catalog was error-free.

### The Solution
I developed and executed a systematic data entry workflow to create a clean, unified product catalog.

1.  **Data Collection:** Methodically gathered all required data points for 200 products, including SKU, Product Name, Category, Description, Price, and Image URL.
2.  **Structured Data Entry:** Entered the collected data into a well-organized Excel spreadsheet with clearly defined columns corresponding to the e-commerce platform's fields.
3.  **Quality Assurance:** Performed a thorough review of the entire dataset to check for typos, formatting errors, and missing information, ensuring 100% data integrity.
4.  **Final Delivery:** Provided the client with a clean, import-ready CSV file and a user-friendly Excel file with data filters for easy management.

### Tools Used
*   **Microsoft Excel:** For data entry, organization, and quality assurance.
*   **Google Sheets:** For potential real-time collaboration with a client.
*   **Web Research / Data Extraction Tools:** For efficient data collection from source pages.

---

### My Process
1.  **Template Creation:** Set up an Excel template with all the required fields (SKU, ProductName, etc.) to ensure consistency from the start.
2.  **Data Gathering:** Systematically navigated through the source materials, copying and pasting or manually entering data into the template.
3.  **Data Formatting:** Ensured uniform formatting across the sheet, such as standardizing currency format (e.g., `135.71` not `$135.71`) and consistent capitalization for categories.
4.  **Validation Check:** Used Excel's filtering and sorting features to quickly spot outliers or missing data. For example, I sorted by price to identify any potential typos (e.g., a price of $1.35 instead of $135).
5.  **Final Export:** Saved the final, verified data as both an `.xlsx` file and a clean `.csv` file for platform import.

---

### Key Outcomes & Results
*   **Speed:** Delivered the complete 200-product catalog in just **5 hours**.
*   **Accuracy:** The final submitted file was **100% error-free**, eliminating the need for any revisions and allowing the client to launch their products immediately.
*   **Organization:** The structured file saved the client's development team an estimated 3-4 hours of data formatting and cleanup work.
